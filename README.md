TMI-Lowdown
--------------------

Twitch chat parser powered by Typescript and Rxjs.

`npm install tmi-lowdown`

Get the lowdown on twitch chat with a simple but powerful interface making ws-irc easy.





Dev
--------------

setup: 
```
  - npm install
  - npm start
```

build and run:
```
  - npm install
  - npm build
  - node index
```

tests:
```
  - npm install
  - npm test
```
